CREBP-Disease-Treatment
=======================
Research to find the gaps in Random Controlled Trials (RCTs) and Systematic Reviews (SRs).


Installation
------------

	git clone https://github.com/CREBP/CREBP-Disease-Treatment.git
	cd CREBP-Disease-Treatment
	npm install
	bower install


Build data
----------

	npm run build


Run basic server
----------------

	npm start
